Project

People Analytics Pattern Catalog

COMPLETE

A comprehensive catalog of 93 Microsoft Fabric analytics patterns across 8 HR domains, with an interactive pattern builder and Streamlit exploration app.

Overview

The People Analytics Pattern Catalog provides a structured, searchable library of analytics patterns designed for Microsoft Fabric environments. Covering 93 patterns across 8 domains — from Workforce Planning and Talent Acquisition to Compensation Analytics and DEI — each pattern includes implementation guidance, KPI definitions, data requirements, and Fabric-specific architecture notes. The project includes an interactive web-based pattern builder for assembling custom analytics solutions, plus a Streamlit app for exploring patterns by domain, complexity, and data source.

Pattern Framework

The framework is organized across eight distinct HR analytics domains, each containing patterns designed to address specific business questions and analytical needs:

8 Analytics Domains: Workforce Planning, Talent Acquisition, Employee Experience, Learning & Development, Compensation Analytics, DEI & Belonging, Workforce Health & Safety, HR Operations

Pattern Structure: Each pattern documents the business question, required data sources, KPIs with calculation logic, Fabric lakehouse architecture, and governance considerations

Interactive Builder: Web-based tool for selecting and combining patterns into custom analytics solutions with dependency tracking

Streamlit Explorer: Python-based app for filtering and browsing patterns with visualization of relationships and prerequisites

Governance Notes

All patterns are based on publicly available Microsoft Fabric documentation and general people analytics domain knowledge. No proprietary organizational data, schemas, or methodologies are referenced. Pattern KPIs use industry-standard calculations.
