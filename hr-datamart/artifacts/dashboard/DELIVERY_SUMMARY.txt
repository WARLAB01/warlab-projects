DEPLOYMENT INSTRUCTIONS
================================================================================

AWS S3 + CloudFront Setup:

1. Upload to S3:
   aws s3 cp index.html s3://your-bucket/
   aws s3 cp error.html s3://your-bucket/

2. CloudFront Configuration:
   - Origin: S3 bucket
   - Default root object: index.html
   - Error handling: Route 404 to error.html
   - Cache behavior: 5 min HTML, 1 hour assets
   - Enable compression & caching

3. Lambda Data Endpoints:
   - Serve /data/*.json from Lambda
   - Enable CORS on Lambda responses
   - Ensure TLS/HTTPS enforcement

4. DNS & HTTPS:
   - CloudFront domain or custom domain
   - ACM SSL certificate
   - Route 53 DNS records

================================================================================
CUSTOMIZATION GUIDE
================================================================================

Changing Colors:
  Edit CSS variables in <style> section:
  --color-primary, --color-highlight, etc.

Modifying Refresh Rate:
  CONFIG.refreshInterval = [milliseconds]

Adding New Charts:
  1. Create chart container in HTML
  2. Create createXxxChart() function
  3. Register in appropriate renderTab()
  4. Add to CONFIG.charts object

Adjusting Demo Data:
  Edit generateDemoData() function:
  - Number of companies/departments
  - Headcount ranges
  - Compensation scales
  - Trend patterns

Data Schema Customization:
  Update fetch logic in loadData()
  Adjust data mapping in render functions
